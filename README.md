# Computer-Networking-Project

## lab1
- A multi-stage client-server protocol using both UDP and TCP sockets
- Constructed and parsed binary packet headers (big-endian, alignment-safe) with payload validation and protocol state enforcement
- Implemented UDP reliability mechanisms, including retransmissions, acknowledgements, and ordering guarantees
- Transitioned between UDP and TCP phases while maintaining protocol correctness and secret-based authentication

## lab2
- Implemented a reliable data transport protocol over UDP, simulating TCP-like behavior, and designed logic for packet sequencing, ACK handling, retransmission timers, and loss recovery
- Analyzed network behavior under packet loss and latency to ensure correctness and efficiency

## lab3
- A multithreaded HTTP proxy server capable of handling multiple simultaneous clients
- Implemented request parsing, forwarding, response handling, and caching, following HTTP/1.1 specifications
- Managed concurrency, shared resources, and connection lifecycles to ensure correctness and scalability
